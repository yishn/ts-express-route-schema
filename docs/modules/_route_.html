Route

  • Route<S>(schema: S, implementations: S extends RouteSchema<infer M> ? MethodImpls<M> : never): Route<S>
    • Defined in src/Route.ts:48

    Defines a route that implements all methods that is defined in the route schema.

    Example

     
    const TestRoute = Route(TestRouteSchema, {
 async get(data) {
   // Do stuff

   return {
     body: {
       message: `Testing, ${data.query.name}`
     }
   }
 }
})

    You can mount the route directly on an Express router with:

    const router = express.Router()

router.all(...TestRoute)
